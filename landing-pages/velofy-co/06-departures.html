Built in Gurgaon. Cleared for everywhere.

Velofy is a small crew of AI Native builders working out of Gurgaon, India, a short drive from the runways of Delhi's IGI. We build for a web where the next reader of your code is as likely to be an agent as a person, and everything we make leaves this terminal as open source.

First off the stand is Summit.js, an open source AI Agent Native JavaScript framework for composing behavior directly in HTML. No build step, no virtual DOM, no eval. CSP-safe interpreted expressions, a fine-grained signal engine, keyed list rendering, and a bundled accessible UI library, all in about 16KB gzipped. The docs are machine-readable too: an llms.txt for context windows and a drop-in AGENTS.md for your repo. More repos are opening soon.

Open Source. by Nature.
