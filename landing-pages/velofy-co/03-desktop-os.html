summit.js

Summit.js

an open source, AI Agent Native JavaScript framework

Summit composes behavior directly in HTML. No build step, no virtual DOM, no eval. Expressions are interpreted and CSP safe, state runs on a fine grained signal engine, lists render keyed, and an accessible UI library ships in the box. All of it in about 16KB gzipped.

Agents are first class readers here: docs ship as llms.txt, and a drop in AGENTS.md teaches your coding agent the whole framework in one file.

hello.html

<div s-data="{ open: false }">
  <button @click="open = !open">Toggle</button>
  <p s-show="open">Hello from Summit</p>
</div>

VELOFY README

version 1.0, written in Gurgaon, India

We are AI Native builders. We think the next generation of software will be written with agents, read by agents, and shipped in the open.

So we make tools that speak plain HTML, publish docs that machines can parse, and never ask you to install a build step before you say hello world.

HOUSE RULES

  1. Ship in the open.
  2. Keep the runtime honest. No eval, no magic.
  3. Write docs for humans and machines alike.
  4. Small is a feature. 16KB is a statement.
