Depth, softly

Glassmorphism

Translucent surfaces that blur and lift what sits behind them. It reads as physical glass floating over a vivid backdrop.

Use for overlays, modals, and cards on colorful or photographic backgrounds. Needs contrast behind it to work.

background: rgba(255,255,255,.14);
backdrop-filter: blur(16px) saturate(1.5);
border: 1px solid rgba(255,255,255,.35);
border-radius: 20px;
box-shadow: 0 8px 32px rgba(20,20,60,.28);
Soft UI
Extruded from the surface

Neumorphism

One background color, two shadows: a light one up-left and a dark one down-right. Elements look pressed out of the same clay. Press the button to invert it.

Use sparingly. It is beautiful but low-contrast, so it fails accessibility fast. Good for a single hero control, risky for whole forms.

background: #e0e5ec;
box-shadow: 9px 9px 18px #a3b1c6,
            -9px -9px 18px #ffffff;
/* pressed: use inset on both */
Puffy and friendly

Claymorphism

Rounded, inflated shapes with a big soft drop shadow and two inset shadows, a dark one at the bottom and a light one at the top. Everything looks squeezable.

Use for playful products, kids apps, and app icons. Pairs well with bright, saturated color.

border-radius: 30px;
box-shadow:
  0 18px 34px rgba(255,107,157,.45),
  inset 0 -8px 14px rgba(155,30,80,.5),
  inset 0 10px 16px rgba(255,255,255,.55);

Ship it raw

Hard borders. Zero radius. One offset shadow doing the whole job.

No apologies

Brutalism

Thick black borders, no rounding, and a solid offset shadow instead of a blurred one. Loud color, monospace, unmistakably made by hand.

Use for portfolios, indie tools, and anything that wants to look confident and anti-corporate. Keep contrast high, it comes for free.

border: 3px solid #000;
border-radius: 0;
box-shadow: 9px 9px 0 #000; /* hard, not blurred */

Ambient light

Aurora mesh

Several big, low-opacity color blobs on a near-black canvas, blurred heavily and drifting on a slow loop. Cinematic, calm, a little hypnotic.

Use for AI and developer landing heroes. Keep it desaturated and slow, and freeze it under reduced-motion.

filter: blur(48px); mix-blend-mode: screen;
animation: drift 26s ease-in-out infinite;
/* 3 blobs, different sizes and delays */

feTurbulence, overlaid

Kills the flat vector look

Grain and noise

An inline SVG turbulence filter laid over a gradient at low opacity with an overlay blend. It adds a film-grain tooth that makes flat color feel printed.

Use for creative studios and editorial pages. Cheap to add, big payoff. Keep opacity under about 0.2.

/* inline SVG data URI */
background-image: url("data:...feTurbulence...");
opacity: .16; mix-blend-mode: overlay;

Modular, asymmetric

Bento grid

A grid of rounded cards in mixed sizes, like a bento box. Each cell holds one idea: a stat, a demo, a quote. It replaced the boring three-column icon row.

Use for feature sections and dashboards. Vary the spans so it does not read as a plain grid, and collapse to one column on mobile.

display: grid;
grid-template-columns: repeat(4, 1fr);
/* cells: grid-column: span 2; grid-row: span 2 */

Early 2000s energy

Retro Y2K

Bevelled chrome made from a vertical gradient that flips from light to dark at the midpoint, soft candy backgrounds, and little sparkles. Peak turn-of-the-millennium.

Use for music, fashion, and nostalgia-driven brands. A knowing wink, not a default.

background: linear-gradient(#fff,#cfd6e6 44%,#8b93ac 52%,#eef2fb);
box-shadow: inset 0 2px 2px rgba(255,255,255,.9),
            inset 0 -3px 4px rgba(60,60,100,.4);
open all night
Light in the dark

Neon glow

Stacked text-shadows and box-shadows in a saturated hue on a near-black ground, tuned to look like gas tubing. The trick is layering several shadows at growing blur radii.

Use for gaming, nightlife, and music. Keep it to one or two hues so it glows instead of smears.

text-shadow:
  0 0 6px #fff, 0 0 16px #ff2ec4,
  0 0 34px #ff2ec4, 0 0 60px #ff2ec4;
make it flow
Color as the subject

Gradient and duotone

A gradient clipped to the shape of the text, so the letters themselves carry the color. The same gradient can tint an image via a blend for a matched duotone.

Use for a hero headline or a single accent. One clipped gradient reads as intentional, five reads as noise.

background: linear-gradient(100deg,#f9c449,#ff5c8a,#8b5cf6);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
Brushed metal
Made of real things

Skeuomorphism

Interfaces that imitate physical materials: stitched leather, brushed metal, a switch with a real bevel. Built from layered gradients plus inset and outset shadows.

Use for a deliberate retro or tactile moment. It is heavy and dated at scale, but a single physical control can delight.

background: linear-gradient(#7fd07f,#3fae52);
box-shadow: inset 0 3px 6px rgba(0,0,0,.35),
            inset 0 -2px 2px rgba(255,255,255,.4);
