For your agent

Every plate is one self-contained HTML file: inline CSS and JS, system fonts, zero requests. Hand a file to a coding agent as a design reference; the tokens live in :root, the markup is semantic, the copy shows the voice. Built in the spirit of Summit.js.

Open Source. by Nature. · MIT · github.com/velofy/ui-atlas · summit docs This index is one HTML file. No JavaScript, no webfonts, no images. It loads the way we build.
